# Yummy_cakes
Cake Business Web Application ; A fully responsive front-end web application for a cake business, designed with a focus on user experience, clear navigation, and efficient ordering flow. Includes product display, categorized menu, and a custom cake request feature. Built using HTML, CSS, and JavaScript.

A responsive cake ordering website 

## Features
- Browse cakes
- Order cakes
- Custom cake request

## Tech Used
- HTML
- CSS
- JavaScript
